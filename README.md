# 🛒 Mercadez - Mercado Nota Dez!

Plataforma web desenvolvida para auxiliar pequenos comerciantes e consumidores na **gestão de estoque** e **comparação de preços** de forma simples, acessível e eficiente.  
O projeto foi criado na FATEC Ferraz de Vasconcelos.

---

## 🚀 Funcionalidades Principais
- Autenticação de usuários (Administrador, Lojista e Cliente).
- Cadastro e edição de produtos.
- Comparação de preços em tempo real entre estabelecimentos parceiros.
- Dashboard de controle para lojistas.
- Exibição pública de preços e promoções para clientes.
- Criação de listas de compras e favoritos (clientes premium).
- Notificações de promoções.
- Gerenciamento de estoque com integração ao caixa (estabelecimentos premium).

---

## 📊 Diagrama de Caso de Uso
O projeto conta com três atores principais:
- **Cliente**: pesquisa produtos, visualiza ofertas e compara preços.
- **Lojista**: gerencia catálogo, preços e estoque, além de publicar promoções.
- **Administrador**: supervisiona o sistema, controla acessos e garante segurança.

*(Diagrama completo disponível no documento do projeto.)*

## 🛠️ Tecnologias Utilizadas
- **Frontend**: Angular  
- **Backend**: Spring Boot  
- **Banco de Dados**: PostgreSQL  
- **Protótipos de telas**: Figma  

---

## 🌐 Hospedagem
A plataforma está planejada para ser hospedada em ambiente web, garantindo acesso contínuo e seguro para comerciantes e clientes.

## 📌 Observações Finais
O **Mercadez** busca promover:
- Competitividade justa entre pequenos estabelecimentos.  
- Transparência nas informações de preços.  
- Melhoria na gestão de microempreendimentos.  
- Incentivo ao uso de tecnologia no comércio local.  

Espera-se que a plataforma fortaleça a economia regional e se torne uma ferramenta prática e eficiente para comerciantes e consumidores.
